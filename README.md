python-3.10.5; Django-4.1;

Installing dependencies:

python -m venv env
pip install -r requirements.txt

Start with docker:
docker run -d --rm -p 8000:8000 --name portal kraken5010/portalapp

Start with docker-compose:
docker-compose up
